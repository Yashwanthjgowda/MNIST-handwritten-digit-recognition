# MNIST CNN Digit Recognizer

A web-based application that recognizes handwritten digits using a Convolutional Neural Network (CNN) trained on the MNIST dataset. The app is built with **Streamlit** for an interactive and user-friendly interface.

## Features
- Draw digits (0-9) directly on the canvas.
- Real-time digit recognition using a trained CNN model.
- Supports easy integration and testing of custom images.
- Lightweight and efficient for quick predictions.
